#  DigiTools Platform

> A powerful and modern platform that provides useful digital tools to simplify everyday tasks.

---

##  Description

DigiTools Platform is a web-based application designed to offer a collection of essential digital tools in one place.
It helps users perform various tasks efficiently without needing multiple applications.

---

##  Technologies Used

*  React
*  Tailwind CSS
*  JavaScript (ES6+)
*  React Toastify

---

##  Features

*  Add to Cart system with real-time updates
*  Remove items from cart functionality
*  Dynamic total price calculation with precision
